# 安全策略

## 范围

Timet 涉及本地推理、原生移动端桥接和离线知识包，请以负责任的方式提交安全问题。

## 报告方式

在专门的安全邮箱建立之前，请尽量使用 GitHub 私密 Security Advisory 提交。如果暂时无法使用，请先开一个不包含敏感利用细节的最小 issue，并说明你需要私密跟进渠道。

## 建议附带的信息

- 受影响的平台和系统版本
- 设备型号
- 复现步骤
- 风险影响说明
- 是否影响本地推理、原生权限、模型打包、知识包分发或提示词 / 会话处理

## 响应目标

- 尽快确认收到报告
- 尽快复现并评估严重性
- 在公开细节前先完成修复或缓解

## 不在范围内

- 没有具体安全影响的模型质量争议
- 与 Timet 无直接关系的纯上游第三方问题
